# go-cloudmulti

The app is a simplified Heroku clone built with Go and React.

It provisions infrastructure programmatically using Pulumi and Go.

Users can deploy static websites and set up virtual machines.

The main files are 
main.go
, 
sites.go
, and 
virtual_machines.go
.

The app uses AWS for hosting and managing resources.

The 
frontend/src/components
 directory contains React components for the web interface.

The app uses Bootstrap for styling and user interface components.

Configuration is managed through environment variables and Pulumi settings.

The app supports creating, updating, and deleting static websites and virtual machines.

The app uses React for the web interface and user interactions.

The app allows users to create and manage static websites and virtual machines through a web interface.

The app uses Pulumi to provision and manage AWS resources programmatically.

The app provides a user-friendly interface for deploying and managing infrastructure.
